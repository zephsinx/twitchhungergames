# Twitch Hunger Games

A Hunger Games simulator with Twitch integration.

<https://hungergames.zephsinx.com>

## About

TwitchHungerGames is a browser-based hunger games simulator where Twitch chat participants compete in a battle royale-style game. Players join by typing `!play` in the connected Twitch channel's chat, and the game simulates events, combat, and eliminations until one winner remains.

## Features

- Twitch integration for player registration via chat commands
- Multiple themes (Noita, Fridge Survival)
- Event-driven gameplay with randomized outcomes
- Leaderboard tracking wins, kills, deaths, and survival stats
- Player avatars (Twitch avatars or default pixel sprites)

## Usage

1. Enter a Twitch channel name in the input field
2. Click "Connect" to connect to the channel
3. Players join by typing `!play` in chat
4. Click "Start Games" to begin the simulation
5. Good luck, tributes!
